Our Model Design

1. Job Skills Extraction

Model Architecture: Neural Network with BERT Embeddings

  • The training was split (80/20) into training and test data for cross validation with 6,354 training samples and 1,589 validation samples
  • BERT model chosen: It was the “largest” BERT model to choose from, consists of 12 transformer layers and 768 hidden embedding sizes. pretrained using text from Wikipedia
  • Entire BERT model will not be used for the classification à word embeddings would be used with custom neural network built to solve the classification problem
  • Using Word Embedding:
    • allow words with similar meaning to have the same representation
    • lower dimensional dense vectors instead of sparse vector (e.g. one hot encoding) - sparse vector makes it difficult for a neural network to learn and exhaust computational power

Transfer Learning

  • Allow the model to be trained on subset of linkedin dataset before applying the model across the full dataset
  • Essential in our project as we could only obtain a full dataset till very late in our project

2. Employee Retention Prediction

  • Logistic Regression: AUC = 0.83
  • Give live prediction on the probability of an employee leaving the company based on employee information

3. Job Title Similarity 

  • Multi-class Text Classification: Department and Responsibilities

4. Salary Distribution Distance

  • Quantify the difference between employee salary distributions and salary distributions in job posts
  • KL divergence score to determine Statistical Distance
  • KL(P || Q) = sum x in X P(x) * log(P(x) / Q(x))
